Snyder, Lawrence: On uniquely represented data strauctures. In: 18th Annual Symposium on Foundations of Computer Science (sfcs 1977) : IEEE, 1977, pp. 142–146

A model for searching algorithms is developed which includes most tree-like searching structures such as lists, binary trees, AVL trees and 2,3-trees. It is shown that no searching algorithm employing a data structure that is uniquely represented (up to isomorphism) can provide search, insert and delete functions all operating faster than cn​ time for every n key tree. The cn​ bound is shown to be achievable for uniquely represented data structures.
